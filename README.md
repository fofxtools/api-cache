# API Cache

A Laravel package for caching and rate limiting API responses. **Currently under development.**

## Overview

This package aims to provide:
- Automatic caching of API responses
- Rate limiting for API requests
- Support for multiple API clients
- Configurable cache TTL and rate limit windows

## Current Status

🚧 **Work in Progress** 🚧

This package is in early development and not ready for production use. Core features are being implemented and tested.

## Requirements

- PHP 8.2+
- Laravel 11.x
- MySQL/MariaDB or SQLite
